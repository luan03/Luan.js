commitar: git commit -m "comentario de alteracao"
update: git pull origin master
subir pro servidor: git push origin master
saber arquivos alterados: git status

adicionar arquivos: git add

tirar arquivos do add: git  reset HEAD nomedoarquivo.php

discartar arquivos: git checkout nomedoarquivo.php (desfazer alterações rollback)

git add -p (mostra oque foi alterado no arquivo a ser comitado)
é mostrado as opçoes (básico, escolha apenas y/n) sim ou não

lista todas as alterações estão no staging =  git diff --staged
